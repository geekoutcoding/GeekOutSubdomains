How to Get Your Subdomain

You can request any available subdomain under:

https://<subdomain>.gosb.hs.vc

To claim one:

1. Fork the official GitHub repository:
github.com/geekoutcoding/infinityfreebios

2. Inside your fork, create a file named domain.md.

3. Add the following required fields:

  • name — your real or preferred name
  • email — used only for verification
  • description — what the subdomain will be used for
  • theme — dark, light, neon, minimal, etc.
  • domain — the subdomain you want (example: josh.gosb.hs.vc)

Example domain.md:

name: Josh
email: example@mail.com
description: Personal project hub.
theme: minimal
domain: josh.gosb.hs.vc

What Happens Next

Once your request is reviewed and approved, your subdomain will be created and pointed to your hosted page.

You can use it for:

  • Personal websites
  • Project pages
  • Developer portfolios
  • Bio link hubs
  • Anything static‑site compatible

Your final URL will look like:

https://yourname.gosb.hs.vc
