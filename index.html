How to Get Your Subdomain

You can request any available subdomain under:

https://<subdomain>.geekout.lol

To claim one:

1. Fork the official GitHub repository:
github.com/geekoutcoding/infinityfreebios

2. Inside your fork, create a file named domain.md.

3. Add the following required fields:

  • name — your real or preferred name
  • email — used only for verification
  • github_pages — please put your GitHub Pages link here
  • domain — the subdomain you want (example: josh.geekout.lol)

Example domain.md:

name: Josh
email: example@mail.com
github_pages: https://yourusername.github.io
domain: josh.geekout.lol

How Verification Works

Once you submit your domain.md, your request will be reviewed manually.

We check:

  • That the subdomain is available
  • That your GitHub Pages link is valid
  • That your email is reachable

If approved, your subdomain will be pointed to your GitHub Pages site using a CNAME record.

Your final URL will look like:

https://yourname.geekout.lol
